Що таке Git?

Git — це розподілена система контролю версій, яка дозволяє працювати з різними версіями проекту, легко відстежувати зміни та співпрацювати над проектами в команді.

Базові команди Git

Нижче наведені найважливіші команди Git для початку роботи:

git init
git clone <url>
git add <file>
git commit -m "опис змін"
git status
git branch
git checkout <branch>
git merge <branch>
git pull
git push
